Repository Overview

One repository, three production surfaces.

The Kumquat repo combines the public website, the protocol implementation, and the AWS-hosted platform needed to run the system. The site for kumquat.info should explain the whole machine, not just present a generic crypto homepage.

Website

Django application

Public pages, auth flow, admin surfaces, crawler operations, and backend endpoints are served from one application and hostname.

Blockchain

Rust chain runtime

Node runtime, storage, consensus, networking, and genesis tooling back the denomination-based money system.

Infrastructure

AWS + k3s + Terraform

Private platform infrastructure covers VPC, VPN, ECR, MySQL, persistent storage, and deployment orchestration.

Cash Model

Money that reads like money.

Kumquat’s core claim is not more finance widgets. It is a different object model for value: large denominations remain discrete, small change stays batch-like, and the interface keeps those units visible.

Hybrid Protocol Direction

Bills behave like owned objects. Coins behave like accountable inventory.

The product is designed around the tactile logic of cash. High denominations are meant to feel countable, movable, and inspectable. Lower-value coins remain fungible below the dollar to keep the system efficient where object-level granularity adds little value.

How It Works

Animation should teach the hierarchy.

The interaction language here is built around arrival, stacking, transfer, and counted units. Motion is used to make the model easier to understand, not to decorate around it.

01 / Define

Denominations are first-class

Large bills arrive as their own units, with labels and distinct rows that imply weight, boundaries, and ownership.

02 / Settle

The chain records state

The Rust runtime, storage, and consensus engine support the underlying settlement and shared state transitions.

03 / Operate

The farm runs the surface

Django, infrastructure tooling, crawlers, and backend services turn the protocol into an operable product and delivery system.

Kumquat Farm
